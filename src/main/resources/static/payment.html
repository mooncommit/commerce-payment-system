Follow Commerce 주문 결제
쇼핑몰 주문 생성 응답으로 결제 정보가 자동 입력되었습니다. 결제창 열기 버튼을 누르면 PortOne 결제가 진행되고, 성공 후 서버 결제 확정 API까지 이어서 호출합니다.
사용 순서
  1. Postman으로 로그인해 액세스 토큰을 발급받습니다.
  2. Postman으로 장바구니에 상품을 담습니다.
  3. Postman으로 주문 생성 API를 호출합니다. 사용할 포인트가 있으면 요청 바디에 함께 담으세요.
    응답에서 portonePaymentId와 포인트 차감 후 실결제 금액을 확인합니다.
  4. 아래 STEP 1에 portonePaymentId와 포인트 차감 후 실결제 금액을 입력하고 결제창 열기를 클릭합니다.
    포인트 전액 결제(실결제 금액 = 0)라면 이 페이지를 건너뛰고 바로 5번으로 이동하세요.
  5. 결제 완료 후 화면에 표시된 portonePaymentId를 복사합니다.
  6. Postman으로 결제 확정 API를 호출합니다. 포인트를 사용했다면 사용 금액도 함께 전달하세요.
1
Postman 주문 생성 응답값 입력 필수 입력
서버가 내려준 portonePaymentId 값 그대로 사용합니다.
포인트 차감 후 실제 PG 결제 금액을 입력하세요. 서버가 검증할 금액과 일치해야 합니다.
2
결제자 정보 기본값 그대로 사용 가능
✅ 결제 완료 : PortOne 콜백을 받았습니다
portonePaymentId : 결제 확정 API에 사용
—
다음 단계 : 위 portonePaymentId를 복사한 뒤, Postman으로 결제 확정 API를 호출하세요.
(예: POST /api/payments/confirm : 팀이 설계한 경로 사용)
