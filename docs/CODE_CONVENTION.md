## Reading Rule

- 전체 컨벤션을 한 번에 읽지 않는다.
- 작업 대상과 직접 관련된 섹션 파일만 읽는다.
- 여러 계층을 함께 수정하는 경우 필요한 섹션 파일만 조합해서 읽는다.
- 테스트 작성/수정은 프로젝트 테스트 스킬 또는 기존 테스트 패턴을 우선한다.
